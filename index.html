Sissejuhatus

See veebileht on loodud selleks, et õpetada veebiarenduse põhialuseid. CSS ja HTMLiga saad luua omaenda veebilehti ja projekte. Mõistad, kuidas töötavad peaaegu kõik veebilehed internetis. HTML ja CSS on esimene samm, kui soovid liikuda edasi JavaScripti ja keerukama veebiarenduse juurde. 

          <!DOCTYPE html> <html lang="et"> <head> <meta /* Näide HTMList */
          charset="UTF-8"> <title>Minu esimene leht</title>
          </head> <body> <h1>Tere tulemast!</h1>
          <p>See on minu esimene HTML-leht.</p> </body>
          </html>

Teooria

HTML (HyperText Markup Language) on veebilehtede ehituskivi — keel, millega kirjutatakse ja struktuuritakse veebilehti. HTML määrab ära, millised osad lehel on pealkirjad, lõigud, lingid, pildid, tabelid jpm. See ei määra välimust (seda teeb CSS) ega käitumist (seda teeb JavaScript), aga ilma HTML-ita poleks ühtegi veebilehte.

Semantika ja Navigeerimine

HTML-i semantika tähendab, et iga elementil on oma kindel tähendus ja roll. Näiteks <header> tähistab päist, <main> peamist sisu, <footer> jalust ning <article>, <section> ja <aside> aitavad sisu loogiliselt jaotada. Semantika kasutamine muudab veebilehed arusaadavamaks nii inimestele kui ka otsingumootoritele. See aitab otsingutulemustes paremini silma paista ja tagab parema ligipääsetavuse abivahendeid kasutavatele inimestele. HTML-i ülesehitus põhineb hierarhilisel struktuuril, kus elemendid võivad sisaldada teisi elemente. Tüüpiline veebileht koosneb päisest, navigeerimismenüüst, põhisisust ja jalusest. Struktuuri korrektsus on oluline, et kasutajad leiaksid kiiresti vajaliku info ning et arendajad saaksid koodi lihtsamalt hooldada. Hea struktuur muudab veebilehe loogiliseks ja järjekindlaks. Navigeerimine HTML-is toimub peamiselt <nav> elemendi kaudu, mis sisaldab linke erinevatele leheosadele või teistele lehtedele. Lingid luuakse <a> elemendi abil, millele saab lisada sihtkoha aadressi. Hästi organiseeritud navigeerimine tagab kasutajatele mugava liikumise veebilehel, aitab vältida segadust ja parandab üldist kasutajakogemust. 

<header>
  <h1>Minu koduleht</h1>
</header>

<nav>
  <ul>
    <li><a href="index.html">Avaleht</a></li>
    <li><a href="teenused.html">Teenused</a></li>
    <li><a href="kontakt.html">Kontakt</a></li>
  </ul>
</nav>

<main>
  <p>Siin on kogu peamine sisu.</p>
</main>

<footer>
  <p>&copy; 2025 Minu veebileht</p>
</footer>

Meedia

HTML võimaldab kasutada erinevaid meediaelemente, et muuta veebilehed visuaalselt ja sisuliselt mitmekesisemaks. Näiteks saab pilte lisada <img> abil, heli <audio> abil ja videoid <video> abil. Lisaks on olemas <iframe>, millega saab manustada teisi lehti või teenuseid, näiteks kaarte ja videoid. Meediaelementide eesmärk ei ole ainult ilu või tähelepanu tõmbamine, vaid ka kasutusmugavuse, õppimise, turunduse ja interaktiivsuse toetamine. Korralikult lisatud meedia loob professionaalse, kaasaegse ja kaasahaarava veebikeskkonna. 

      
<img src="https://via.placeholder.com/300" alt="Näidis pilt">

<audio controls>
  <source src="sample-audio.mp3" type="audio/mpeg">
  Teie brauser ei toeta helifaili esitust.
</audio>

<video controls width="400">
  <source src="sample-video.mp4" type="video/mp4">
  Teie brauser ei toeta videofaili esitust.
</video>

<iframe width="400" height="225" 
        src="https://www.youtube.com/
        title="YouTube video"></iframe>

CSS alused

CSS ehk Cascading Style Sheets on tööriist, millega saab kujundada HTML-is loodud struktuuri. CSS eraldab sisu ja kujunduse, mis tähendab, et <h1> võib olla nähtavalt suurem ja värvilisem kui tavaline <p>. CSS abil on võimalik muuta fontide suurust, värvi ja stiili, lisada taustavärve või -pilte, määrata paigutusi ning kohandada elemente, nagu <div>, flexboxi või grid-süsteemiga. Näiteks saab <section> element paigutada ekraanil keskele, samal ajal kui <aside> liigub küljele. Oluline osa CSS-ist on ka responsiivsus. Meediapäringud võimaldavad veebilehe paigutust kohandada vastavalt seadmele – olgu see <header> mobiilis kitsam või <nav> muutumas hamburger-menüüks. Lisaks saab luua dünaamilisi efekte: <a> lingile võib lisada hover-efekti, <button> saab muuta värvi, kui kasutaja sellel klõpsab, ja <img> võib aeglaselt tuhmuda üleminekuga. CSS annab võimaluse muuta iga veebileht ühtaegu funktsionaalseks ja visuaalselt atraktiivseks. 

<style>
  h1 {
    color: #2a9d8f;
    font-size: 36px;
    text-align: center;
  }
  p {
    color: #264653;
    font-size: 16px;
    line-height: 1.6;
  }
  a {
    color: #e76f51;
    text-decoration: none;
  }
  a:hover {
    text-decoration: underline;
    color: #f4a261;
  }
</style>

Kohanduvus ja ligipääsetavus

Kohanduvus HTML-is tähendab, et veebileht töötab erinevatel seadmetel, nagu telefonid, tahvelarvutid ja arvutid. Selle saavutamiseks kasutatakse responsiivset disaini, näiteks CSS-i media queries ja paindlikke paigutusi <div> või <section> elementide jaoks. Ligipääsetavus tagab, et kõik kasutajad, sealhulgas erivajadustega inimesed, saavad sisu hõlpsasti tarbida. Selleks kasutatakse semantilisi elemente nagu <header>, <nav>, <main> ja <footer>, tekstialternatiive piltidele (<img alt>), piisavat kontrasti ning õigesti struktureeritud pealkirju (<h1>–<h6>). Kohanduv ja ligipääsetav veebileht parandab kasutajakogemust, võimaldab kiiret navigeerimist ja suurendab veebilehe kasutatavust kõigile külastajatele. 

<style>
  /* Põhistiilid */
  body {
    font-family: Arial, sans-serif;
    line-height: 1.5;
    margin: 0;
    padding: 20px;
  }

  nav a {
    margin-right: 15px;
    color: #0077cc;
    text-decoration: none;
  }

  nav a:hover {
    text-decoration: underline;
  }

  /* Kohanduvus mobiilseadmetele */
  @media (max-width: 600px) {
    nav {
      display: flex;
      flex-direction: column;
    }

    nav a {
      margin: 5px 0;
    }

    body {
      padding: 10px;
    }
  }
</style>
